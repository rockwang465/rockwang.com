第53讲: css的样式重置
html元素默认会有一些样式，可能会影响我们的布局。
可以去百度搜索: reset:css 会有一堆样式重置的例子，拷贝过来就可以了

例如: 还有很多类似下面这些的设置
/*css样式重置*/
body, p, ul, li, input {  /*将margin padding重置为0*/
    margin: 0;
    padding: 0;
}

  li {  /*取消列表前面的小点*/
    list-style: none;
}

  a {   /*取消下划线*/
      text-decoration: none;
  }

  .pull-left {  /*将盒子浮动到左侧*/
      float: left;
  }

  .pull-right {  /*将盒子浮动到右侧*/
      float: right;
  }

  .clearfix {  /*兼容IE的hasLayout*/
      *zoom: 1;
  }
  .clearfix:after {  /*清除浮动异常问题*/
      content: ".";
      display: block;
      height: 0;
      visibility: hidden;
      clear: both;
  }
